# Customer Churn Analysis

## Project Overview
This project focuses on analyzing customer churn behavior to identify patterns and factors contributing to customer attrition. The insights can help businesses improve customer retention strategies.

## Business Objectives
- Calculate overall churn rate
- Analyze churn across contract types
- Understand impact of monthly charges on churn
- Analyze tenure-based churn behavior
- Identify high-risk customer segments

## Dataset
- Telecom customer churn dataset

## Tools & Technologies
- Python (Pandas)

## Key Analysis Performed
- Cleaned data by converting TotalCharges to numeric format
- Handled missing values
- Performed exploratory data analysis (EDA)
- Analyzed churn patterns by contract, tenure, and pricing

## Key Insights
- Month-to-month contract customers have the highest churn
- Customers with higher monthly charges churn more frequently
- Customers with shorter tenure are more likely to churn
